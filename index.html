BasePulse

A pay-per-call data API for AI agents on Base. Agents pay $0.001 in USDC per request via the x402 protocol — no API keys, no subscriptions.

Endpoints

  • GET /gas — current Base gas price
  • GET /block — latest block number + timestamp
  • GET /balance/:address — ETH balance of any address

Built with x402 + Base Builder Codes.
